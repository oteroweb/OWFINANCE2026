Bottom Navigation Component Details

Component Name: LiteBottomNav / MobileBottomNav

Location: src/components/LiteBottomNav.vue

Purpose: Primary navigation for Lite version with 4 main tabs and floating action button

Features

  • 4 main navigation tabs (Home, Transactions, Jars, Settings)
  • Active tab indicator (underline)
  • Floating Action Button (FAB) for quick add transaction
  • Safe area padding for notch/home indicator
  • Icon + label per tab

Props

activeTab: String (required, one of: 'home', 'transactions', 'jars', 'settings') tabs: Array (Array of tab objects with id, icon, label) fabIcon: String (default: '+') showFab: Boolean (default: true)

Events

@tab-changed: { tabId: String } Emit when tab is clicked @fab-clicked: Emit when FAB is clicked

Safe Area Compliance

padding-bottom: env(safe-area-inset-bottom) Handles iPhone notch and home indicator Works on all mobile devices
