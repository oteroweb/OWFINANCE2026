Pro Topbar Component Details

Component Name: ProTopbar / DesktopTopbar

Location: src/components/ProTopbar.vue

Purpose: Top navigation bar for Pro desktop version with search, notifications, and utility menu

Key Features

  • Logo with branding icon
  • Breadcrumb navigation (optional)
  • Global search functionality
  • Notification bell with badge counter
  • Settings/utility actions
  • User avatar with dropdown menu
  • Sticky positioning

Props

logoText: String (default: 'OWFinance') logoIcon: String (default: '💰') breadcrumbs: Array (optional) showSearch: Boolean (default: true) notificationCount: Number (default: 0) userInitials: String userName: String searchPlaceholder: String

Events

@search: { query: String } @notification-click: Emit when notification bell clicked @settings-click: Emit when settings icon clicked @avatar-click: Emit when user avatar clicked

Responsive Design

Desktop (> 1200px): Full topbar with search and breadcrumbs Tablet (900px - 1200px): Hide breadcrumbs, reduce search width Mobile (< 900px): Hide search, compact layout
