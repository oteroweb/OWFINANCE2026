JarCard

src/components/JarCard.vue

Displays a single jar card with balance, budget info, and quick stats. Supports accumulative/reset modes with live preview of percentage changes.

Jars

Props

Name Type Default Description
jarRequired Record — Jar object with name, type, percent, fixedAmount, refresh_mode, active
balance object | null null Balance data: asignado, gastado, ajuste, retiros, transfers, leverage
loading boolean false Shows loading overlay when true
error string | null null Error message banner
showLeverage boolean false Show leverage button
useRealIncome boolean false Use real income (true) or expected income (false)
expectedIncome number 0 User-configured expected income
calculatedIncome number 0 Calculated/real income from transactions

Emitted Events

Event Payload Description
adjust void User clicked adjust button
withdraw void User clicked register usage button
leverage void User clicked leverage button
reset void User clicked reset adjustment

Key Features

  • Real-time balance display with color-coded progress bar
  • Expandable detailed breakdown with full transaction history
  • Multi-state indicators (accumulative/reset, positive/negative, over-budget)
  • Live preview when jar percentage changes
  • Carry-over amount tracking for accumulative jars
  • Quick action buttons (Adjust, Withdraw, Leverage, Reset)
  • Responsive design for mobile/desktop
  • Dark mode support via CSS variables

Example Usage
<template>
  <JarCard
    :jar="jarData"
    :balance="jarBalance"
    :loading="isLoading"
    :expected-income="expectedIncome"
    :calculated-income="realIncome"
    :use-real-income="false"
    @adjust="onAdjustJar"
    @withdraw="onWithdraw"
    @leverage="onLeverage"
  />
</template>

<script setup lang="ts">
const jarData = {
  id: 1,
  name: 'Gastos Generales',
  type: 'percent',
  percent: 30,
  refresh_mode: 'accumulative',
  active: true
};

const jarBalance = {
  asignado: 1500,
  gastado: 975,
  ajuste: 0,
  balance: 525,
  transfers_in: 0,
  transfers_out: 0,
  leverage_in: 0,
  leverage_out: 0,
  saldo_anterior: 100
};
</script>

JarsBalanceBar

src/components/JarsBalanceBar.vue

Horizontal scrollable bar showing quick summary of all jar balances. Color-coded by status (ok, warning, negative).

Jars

Key Features

  • Auto-loads jar balances on mount and period change
  • Horizontal scrolling for many jars
  • Status chips: ok (green), warning (orange), negative (red)
  • Manual refresh button with loading state
  • Period label with short month/year format
  • Responsive design collapses names on mobile
  • API: GET /jars/all-balances?year=Y&month=M

ExpenseDistributionChart

src/components/ExpenseDistributionChart.vue

Dual ECharts visualization: pie chart showing budget distribution and bar chart comparing assigned vs spent vs balance.

Charts

Props

Name Type Description
rowsRequired JarChartRow[] Array of jar data with id, name, color, assignedExpected, spent, balance
currencyCodeRequired string Currency code for formatting (e.g., 'USD', 'MXN')
hideValues boolean When true, shows '••••••' instead of amounts for privacy

Key Features

  • Interactive pie chart: budget distribution by jar
  • Interactive bar chart: assigned vs spent vs balance comparison
  • Responsive grid layout: adapts to screen width
  • ECharts with Canvas renderer for performance
  • Currency formatting with Intl API fallback
  • Value obfuscation for privacy-sensitive contexts
  • Custom tooltips with detailed information
  • Scrollable legend for many jars
