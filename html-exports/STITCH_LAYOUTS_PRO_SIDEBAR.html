Pro Sidebar Component Details

Component Name: ProSidebar / DesktopSidebar

Location: src/components/ProSidebar.vue

Purpose: Left sidebar navigation for Pro desktop version with drag & drop reordering and user profile

Key Features

  • Primary navigation items (Home, Transactions, Jars, Analytics)
  • Secondary management section (Accounts, Categories)
  • Drag & drop reorderable items
  • Active state indicator with left border
  • Settings link in footer
  • User profile card with avatar, name, and email
  • Collapsible sections (future)

Props

items: Array (required, navigation items) activeItem: String (required, active item id) userInitials: String userName: String userEmail: String enableDragDrop: Boolean (default: true) collapsibleSections: Boolean (default: false)

Events

@item-click: { itemId: String } @items-reordered: { newOrder: Array } @user-profile-click: Emit when user profile clicked @settings-click: Emit when settings clicked

Drag & Drop Implementation

Uses: @vue-dnd/core or similar library Draggable: true via drag handle (⋮) Drop zones: Main navigation, Management section Persistence: Save to localStorage Events: Emit on reorder completion
